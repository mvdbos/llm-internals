# Private case-study metadata

This directory is intentionally excluded from the public course except for this README.

Private manifests may contain exact checkpoint identifiers, provider names, local filesystem paths, prompts, raw job-to-task mappings, patches, and trajectory locations. None of those values may be copied into public lessons, public JSON, HTML results, or commits.

Public artifacts must use stable aliases for checkpoints, precision conditions, tasks, and repetitions. They may disclose architecture class, parameter class, backend and revision, quantization mode and group size, controlled settings, attrition categories, and sanitized trial metrics.
