High-trust evaluations

  • The case for 4-bit precision — ICML 2023. Shows that block size, numeric type, model scale, and bit width jointly affect the trade-off.
  • Evaluating Quantized Large Language Models — ICML 2024. Separates weight, activation, and KV-cache quantization across task types.
  • A Comprehensive Evaluation of Quantization Strategies for LLMs — Findings of ACL 2024. Perplexity correlates with many but not all downstream outcomes; memory savings do not guarantee speedups.
  • Quantization Hurts Reasoning? — COLM 2025. Effects depend on model, size, target, bit width, and task; extra reasoning can help some quantized models.
  • Unified llama.cpp quantization evaluation — 2026 preprint. A useful controlled single-model GGUF comparison, not peer-reviewed or universal.
